# FitMaker Gym Website

A modern, responsive gym website for **FitMaker**, built with **React**, **Vite**, and **Tailwind CSS**.  

---

## Features

- Fully responsive design  
- Workout plans and training steps  
- Machines and equipment sections  
- Trainer profiles and guidance  
- Smooth UI animations  

---

## Installation

1. Clone the repository:
git clone https://github.com/yourusername/fitmaker-gym-website.git

2. Navigate to the project folder:
cd fitmaker-gym-website

3. Install dependencies:
npm install

4. Start the development server:
npm run dev

done!
